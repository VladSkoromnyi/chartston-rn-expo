# chartston

GPU-accelerated interactive candlestick charting library for React Native / Expo (New Architecture, Skia + Reanimated)

## Installation


```sh
npm install chartston
```


## Usage


```js
import { multiply } from 'chartston';

// ...

const result = multiply(3, 7);
```
